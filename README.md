Tytul: Informatyka-projekt

Opis scenariusza:
- nalezy skompilowac program
- w MENU porusza sie strzalkami gora/dol
- wybrac "nowa gra" nastepnie potwierdzic ENTER
- celem gry jest zniszczenie wszystkich cegiel - czerwonych, zoltych oraz niebieskich kolejno 1 zycie, 2 zycia oraz 3 zycia
- gdy gracz przegra po chwili jest cofany do okna MENU
- po wygranej zadzieje sie to samo
- mozna zapisac swoj postep tworzac zapis klikajac klawisz F5, po przegraniu aby powrocic to zapisanego zapisu nalezy wybrac opcje "wczytaj gre"
- aby wyjsc z gdy nalezy wybrac opcje "wyjscie"


Spelnienie wymagan :
- zapis/wczytanie postepu
- animacja poruszania sie paletki, pilki, odliczania(dynamiczny tekstowy) oraz wybuchu paletki (nieregularny ksztalt)
- mechanizm kolizji
- obsluga zakonczenia programu
- sterowanie z klawiatury
- struktury danych i klasy
